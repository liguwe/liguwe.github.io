
# Raycast 使用技巧总结

<Badge type="info">#工具</Badge> <Badge type="info">#工具技巧</Badge> <Badge type="info">#常用工具</Badge> <Badge type="warning">#raycast</Badge>  <Badge type="warning">#2023/11/26</Badge>

- 支持模糊搜索
	- 如输入 `vs code` 可以找到 Visual Studio Code 应用程序
	- 可以设置 模糊搜索 `灵敏度`
- 设置别名 Aliases
	- 比如，设置 `飞` 带别 `飞书`
- 全局快捷键 Hotkeys
	- 可以单独设置某个具体操作的全局快捷键
- Action Panel 命令面板
	- `⌘ K` 用于展示更多命令
	- `⌘ ↵` 用于提交表单
	-  `↵`  用于执行操作
- 代码片段
	- 可以设置常用的片段，如代码、常用的输入文本
	- 比如输入，email 显示 `liguwe`
- Script Commands 脚本命令
	- 可实现任何的脚本命令控制电脑

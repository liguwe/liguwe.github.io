# 生成个人博客网站

- 基于`vitepress` 搭建
- 支持一些 `vitepress` 语法，具体可看 [官网](https://vitepress.vuejs.org/guide/getting-started.html)
- 数据来源于 Obsidian App

## 本地开发

使用 node20 版本

`pnpm install`

`npm run dev`


## 部署到 liguwe.github.io

`sh publish.sh`

## 发布到这三个地方

`npm run publish`

- liguwe.github.io
- yuque.com/liguwe/post
- yuque.com/liguwe/obsidian

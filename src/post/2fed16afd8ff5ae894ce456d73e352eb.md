
# CSS 处理器都有哪些？


#css  #前端/CSS  #2024/07/30 

又分为 `前置处理器（预处理器）`和 `后置处理器（后处理器）`

## 1. CSS 前置处理器（预处理器）

- Sass/SCSS
	- Sass (Syntactically Awesome Style Sheets)
- Less (Leaner CSS)
- Stylus

## 2. CSS 后置处理器（后处理器）

代表

- PostCSS（虽然它也可以作为预处理器使用）
- Autoprefixer（自动添加浏览器前缀）
- CSS Nano（压缩和优化 CSS）

## 3. CSS-in-JS

将 CSS 写入 JavaScript 代码中的技术，比如 styled-components、Emotion、JSS 等。


# 使用 Babel 重构或迁移前端工程

`#前端研效` `#babel` `#前端工程化`

> Babel 不仅可以转换单个文件，还能够处理整个项目的代码结构，并生成新的目录结构和代码。这在项目重构、代码迁移等场景下特别有用

- 目的：项目重构、代码迁移
- 步骤
	- **目录扫描**
		- 递归遍历源代码目录
		- 识别 JavaScript/TypeScript 文件
	- **代码分析**
		- 解析 AST
		- 提取依赖关系
		- 构建依赖图
	- **代码转换**
		- 应用 Babe l转换
			- 使用 babel 来处理 JS 文件
		- 保持目录结构
		- 生成新的代码文件
	- **结构维护**
		- 保持原有的目录层次
		- 维护模块间的依赖关系

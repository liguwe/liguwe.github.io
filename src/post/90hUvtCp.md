 ## 什么是 HTML？ 

- HTML 的全名是“超文本标记语言”（HyperText Markup Language），可以说有了它，才有互联网的**内容**。
- 它是用于构建网页的基本语言，用于定义网页的结构和内容
- 任何网站的请求，第一个请求回来的资源就是`.html` 文件。

## HTML 基本结构

```html
<!DOCTYPE html>
<html>
  <head>
    <title>页面标题</title>
  </head>
  <body>
    <h1>欢迎</h1>
    <p>这是一个段落。</p>
  </body>
</html>
```

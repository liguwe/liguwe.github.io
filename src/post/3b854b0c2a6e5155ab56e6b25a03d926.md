
# CSS Houdini API


#前端/CSS  #2024/07/30 

- **描述**：一组低级 API，允许开发者扩展 CSS 的功能。
- **特点**：可以创建自定义的 CSS 属性、值和渲染逻辑。
- **示例**：使用 Houdini Paint API 创建自定义背景图案。

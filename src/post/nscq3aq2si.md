
# 什么是 NPC ？

`#2025/12/25`

**NPC = Non-Player Character（非玩家角色）**


![image.png|483](https://832-1310531898.cos.ap-beijing.myqcloud.com/202520251225120828439.png)


**一句话**：游戏里不是由玩家控制的角色，全部由电脑/程序控制。

```bash
《塞尔达传说》：
├─ 你控制的林克 → 玩家角色
└─ 其他所有角色 → NPC
   ├─ 塞尔达公主（剧情角色）
   ├─ 村民（提供任务/对话）
   ├─ 商人（卖东西）
   └─ 怪物（敌人）
```

![image.png|478](https://832-1310531898.cos.ap-beijing.myqcloud.com/202520251225120850195.png)


**NPC = 游戏里除了你之外的所有角色**
- 由程序控制
- 让游戏世界"活起来"
- 包括盟友、敌人、路人

**就像电影里**：
- 你 = 主角（亲自演）
- NPC = 配角+群演（导演安排好的

 ## 基本对比 

| 特征   | DeepSeek LLM v3 | DeepSeek R1 |
| ---- | --------------- | ----------- |
| 发布时间 | 2024年2月         | 2024年3月     |
| 定位   | 通用大语言模型         | 专注科研和学术的模型  |

## 总结

- DeepSeek LLM v3 是一个全能型选手，适合大多数使用场景
- DeepSeek R1 是专门为科研人员打造的专业工具
- 两个版本可以互补使用，根据具体任务选择合适的模型

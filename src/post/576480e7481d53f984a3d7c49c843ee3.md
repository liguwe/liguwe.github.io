
# React 源码结构说明

`#React` 

```bash
/packages
  /react                     # React 核心包
    /src
      /React.js             # 入口文件
      /ReactElement.js      # createElement 实现
      /ReactHooks.js        # Hooks 实现
  
  /react-dom                # DOM 渲染器
    /src
      /client              # 客户端渲染相关
      /server              # 服务端渲染相关
      /events              # 事件系统
  
  /react-reconciler        # 协调器核心
    /src
      /Fiber.js           # Fiber 结构
      /ReactFiberBeginWork.js    # render 阶段
      /ReactFiberCommitWork.js   # commit 阶段
  
  /scheduler              # 调度器
    /src
      /Scheduler.js       # 调度算法
      /SchedulerPriorities.js    # 优先级相关

```
